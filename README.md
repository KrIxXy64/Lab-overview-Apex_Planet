# Cybersecurity Pentesting Lab

This repository contains my personal cybersecurity lab setup used for practicing
web, API, and network security testing.

## Lab Overview
- Attacker Machine: Kali Linux
- Vulnerable Apps:
  - DVWA (Manual install)
  - OWASP Juice Shop (Docker)
  - OWASP crAPI (Docker Compose)
- Proxy Tool: Burp Suite Pro
- API Testing: Postman, curl
- Packet Analysis: Wireshark

## Objectives
- Learn cybersecurity fundamentals
- Practice OWASP Top 10 vulnerabilities
- Perform API security testing
- Build hands-on pentesting experience
